# moex0 Blog

Personal cybersecurity blog featuring threat intelligence, incident response case studies, and security research.

## Live Site

**[moex01.github.io/moex0-blog](https://moex01.github.io/moex0-blog)**

## Features

- **Dark terminal-inspired theme** with JetBrains Mono font
- **Auto-generated OG images** for social sharing
- **Full-text search** powered by Pagefind
- **Table of contents** with active section highlighting
- **Reading progress indicator** and back-to-top button
- **Threat level badges** for article severity classification
- **Responsive design** optimized for all devices

## Tech Stack

- [Astro](https://astro.build) 5.x
- [Tailwind CSS](https://tailwindcss.com) v4
- TypeScript
- GitHub Pages

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
